# AI Education Platform

## 项目介绍
AI教育平台是一个综合性的在线教育系统，结合AI技术提供个性化学习体验。

## 项目结构
```
aiedu.old/
├── frontend/        # 前端项目
├── backend/         # 后端项目
└── docs/           # 项目文档
```

## 开发指南
1. 克隆项目
```bash
git clone https://github.com/mhx0628/ai-education-platform.git
cd ai-education-platform
```

2. 环境要求
- Node.js >= 14
- Python >= 3.8
- MySQL >= 8.0
